# Inconsax Svelte

An icon library for svelte.

## Installing

```sh
npm install iconsax-svelte 
```

```sh
bun add iconsax-svelte 
```

## Usage

```svelte
<script lang="ts">
  import Check from "iconsax-svelte/bulk/Check.svelte";
</script>

<Check color="red" />
```
