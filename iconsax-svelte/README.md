# Inconsax Svelte

This template should help get you started developing with Svelte and TypeScript in Vite.

## Installing

```sh
npm install iconsax-svelte 
```

```sh
bun add iconsax-svelte 
```

## Usage

```svelte
<script lang="ts">
  import Iconsax from "iconsax-svelte";
</script>

<Iconsax icon="activity" />
```
